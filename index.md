接口鉴权: 否  
测试地址: https://api.baojie.ink/login  
正式地址: https://api.baojie.ink/login  
请求方式: POST  
请求参数:
```json
{
	"phone":"18585131312",
	"password":"142536"
}
```

```json
{
    "code": 0,
    "msg": "成功"
}
```
